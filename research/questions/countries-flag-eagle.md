# countries-flag-eagle

**Theme:** countries
**Date:** 2026-05-04

## Original
> Quels pays ont un aigle sur leur drapeau ?

## Disambiguated
> Pays dont le drapeau national officiel comporte un aigle (aigle simple, aigle bicéphale, aigle de Saladin, aigle pêcheur). Exclut les autres rapaces et oiseaux (frégate, paradisier, grue, perroquet). Inclut les aigles dans les armoiries portées sur le drapeau officiel.

## Source
- Sous-ensemble de countries-flag-animal.md
- Vérification visuelle des drapeaux sur Wikipedia

## Raw output
- Albanie : aigle bicéphale (élément principal, fond rouge)
- Égypte : aigle de Saladin (armoiries centrales dorées)
- Kazakhstan : aigle royal (sous le soleil)
- Mexique : aigle royal avec serpent (armoiries, très visible)
- Moldavie : aigle sur armoiries (drapeau tricolore officiel)
- Monténégro : aigle bicéphale sur armoiries (drapeau rouge officiel)
- Serbie : aigle bicéphale sur armoiries (drapeau tricolore officiel)
- Zambie : aigle pêcheur africain (coin supérieur droit, en vol)

Exclusions : Allemagne (Bundesadler uniquement sur les armoiries, pas sur le drapeau civil), Autriche (même situation), Pologne (aigle uniquement sur les armoiries, pas sur le drapeau civil standard).

## Normalization decisions
Tous correspondent directement à des entrées de l'answerSet.

## Final answer set
Albanie, Égypte, Kazakhstan, Mexique, Moldavie, Monténégro, Serbie, Zambie
